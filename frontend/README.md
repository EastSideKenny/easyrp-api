# easyrp-web

EasyRP - From invoice to online store, simplified.

A Nuxt 4 web application using [Bun](https://bun.sh) as the package manager.

## Setup

Install dependencies:

```bash
bun install
```

## Development

Start the development server at `http://localhost:3000`:

```bash
bun run dev
```

## Production

Build the application:

```bash
bun run build
```

Preview the production build:

```bash
bun run preview
```

Check the [Nuxt deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
